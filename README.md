# Project Axiom

> A third-person action gameplay programming project developed in Unreal Engine 5.8 using C++ and Blueprints.

## Overview

Project Axiom is a long-term gameplay programming portfolio project focused on building a clean, scalable gameplay framework using Unreal Engine 5.8.

The project serves as a practical environment for designing, implementing, and refining gameplay systems while following modern Unreal Engine development practices. Development emphasizes maintainable C++ code, modular gameplay architecture, and seamless integration with Blueprints.

As the project evolves, additional gameplay systems will be implemented and documented within this repository.

---

## Objectives

The primary goals of Project Axiom are to:

- Develop gameplay systems using Unreal Engine C++
- Build reusable and maintainable gameplay architecture
- Strengthen gameplay programming and software engineering skills
- Explore production-oriented development workflows
- Create a polished portfolio project for game programming positions
